# 🤖 GPT Bot — Multi-model AI Telegram Bot

Telegram-бот с доступом к 15+ AI-моделям через единый интерфейс. Доступ к более мощным моделям открывается в зависимости от срока VPN-подписки пользователя — монетизация AI через существующую базу клиентов.

## Возможности

### AI модели
- **15+ моделей** в одном боте: GPT-5, Grok 4, DeepSeek R1, Qwen3 235B, Mistral, Gemini и другие
- **Тирная система** — чем дольше пользуешься VPN, тем мощнее доступные модели
- **Переключение модели** — прямо в чате одной кнопкой
- **Поддержка изображений** — отправь фото + вопрос для vision-запросов

### Режимы общения
- 💬 **Обычный** — стандартный диалог
- 🌍 **Переводчик** — RU ↔ EN автоматически
- 💻 **Программист** — ответы с примерами кода в markdown
- ✍️ **Копирайтер** — продающие тексты и посты
- 📊 **Аналитик** — структурированный анализ по пунктам
- 📚 **Учитель** — простые объяснения сложных тем

### Прочее
- **История диалога** — контекст сохраняется в SQLite
- **Сброс контекста** — `/new` для нового разговора
- **Интеграция с VPN-ботом** — проверка подписки через общую БД
- **Заблокированные модели** — красиво показывает что нужно для разблокировки

## Стек

| Компонент | Технология |
|---|---|
| Язык | Python 3.11+ |
| Telegram | aiogram 3.7+ |
| AI API | aitunnel.ru (прокси к OpenAI, Anthropic, xAI, Mistral) |
| База данных | SQLite + aiosqlite |
| HTTP | httpx |

## Установка

```bash
git clone https://github.com/kurumi-mProject/gpt-bot.git
cd gpt-bot
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
nano .env
python bot.py
```

## Переменные окружения (.env)

| Переменная | Описание |
|---|---|
| `GPT_BOT_TOKEN` | Токен от @BotFather |
| `ADMIN_ID` | Telegram ID администратора |
| `AITUNNEL_KEY` | API ключ [aitunnel.ru](https://aitunnel.ru) |

## Доступные модели по уровням

| Уровень | Требование | Модели |
|---|---|---|
| 🆓 Пробный | Любой | GPT-5 Nano, GLM-4.7 Flash, Gemini 3.1 Flash Lite |
| 📅 Базовый | 1+ мес VPN | GPT-4.1 Nano, Qwen 3.5 Flash, MiniMax M2.5 |
| 📅 Стандарт | 3+ мес VPN | GPT-5 Mini, Qwen3 235B, DeepSeek V3.2, Mistral Medium |
| 📅 Продвинутый | 6+ мес VPN | Qwen 3.5 35B, DeepSeek R1, Grok 4.1 Fast |
| 🏆 Максимум | 12 мес VPN | Grok 4 Fast |

## Команды

| Команда | Описание |
|---|---|
| `/start` | Главное меню |
| `/new` | Сбросить контекст диалога |
| `/model` | Сменить AI-модель |
| `/help` | Справка |

## Структура проекта

```
gpt-bot/
├── bot.py          # Точка входа, инициализация
├── config.py       # Конфиг: модели по тирам, режимы, системные промпты
├── handlers.py     # Все хендлеры: команды, сообщения, callback
├── ai.py           # Клиент к aitunnel API (chat completions + vision)
├── database.py     # SQLite: история диалогов, настройки пользователя
└── requirements.txt
```
